# LXVC render engine

Some functionality still WIP, but almost ready to ***MVP***...

## Features

- Minimal and extensible interface (Open Source, SDK)
- Lightweight renderer as prior
- Some automatizations (smart)
- Ray tracing support
- Minimal requirements
- Vulkan API 1.3
